\doxysubsection{Detailed Description}
class implementing the \doxylink{classMapperLib_1_1Mapper}{Mapper} algorithm 

\doxylink{classMapperLib_1_1Mapper}{Mapper} is a data visualization algorithm. It takes a point cloud, projects it to a lower dimension, covers the remaining space in overlapping hypercubes and clusters the original data in each of these hypercubes. Overlapping clusters are now linked by simplices in accordance to the chosen complex. The resulting simplicial complex is the output of the algorithm.
