\doxysubsection{Detailed Description}
Class for generating Cech complexes from overlapping clusters. 

This class implements the \doxylink{classMapperLib_1_1Complex}{Complex} interface and provides methods to generate a cech complex on the given clusters containing simplices up to a specified dimension
